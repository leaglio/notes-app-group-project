# PPE Guard AI

PPE Guard AI is an AI-based workplace safety monitoring system that uses computer vision to detect workers and evaluate PPE compliance from CCTV or video input.

## 🚀 Features
- Worker detection (YOLO)
- PPE compliance checking
- Real-time monitoring
- Violation logging
- Evidence snapshot
- Dashboard system
